🌡️ Clima no Lab API

Bem-vindo à API da plataforma Clima no Lab.

Este serviço fornece acesso aos dados coletados por sensores ambientais utilizados em experimentos e monitoramento de laboratório.

🧪 Sobre o projeto

O Clima no Lab é uma plataforma experimental de aquisição e observação ambiental em laboratório.

Esta API é responsável por gerir os dados coletados para dashboards, interfaces web e outras aplicações.
